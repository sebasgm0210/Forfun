# Casa Luna Boutique Backoffice - React + Vite

Conversión del frontend original de Next.js a React + Vite, manteniendo Tailwind v4, shadcn/Radix, layout, dashboard y navegación.

## Levantar proyecto

```bash
npm install
npm run dev
```

URL normal de Vite:

```txt
http://localhost:5173
```

## Importante

Este proyecto usa Tailwind CSS v4 con `@tailwindcss/vite`. No lo bajes a Tailwind v3 porque se pierden tokens de diseño como `@theme inline`, `@custom-variant`, `bg-sidebar`, `text-muted-foreground`, etc.
